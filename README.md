# Internet of Things
## Week One
Read two sensors and send the data to two cloud services via the NodeMCU.

### Sensors
I used two sensors.

1. **Light Sensor:** Detects the amound of light in my bedroom.
2. **Tilt Sensor:** Detects if the window is open or closed.
